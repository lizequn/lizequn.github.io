Custom Hexo Next Theme

Posted on 09-05-2020 Edited on 31-07-2020

Next theme configuration

Edit Theme config at **./themes/next/_config.yml.**.

  • set scheme to Mist

  • enable tags and about page

  • create tags and about page
  • edit tags page for auto-generate tag cloud

  • enable mathjax

  • swich md render to support mathjax

  • enable gitalk comments

Animation Settings

The default next theme animation is a little bit slow. Speed up it by editing duration variable in ./themes/next/source/js/motion.js.
